# 软件使用说明书 · 文档大纲（母版模板）

安装后复制到项目 `docs/`，按 Manual Contract `manual.doc_path` 命名。

## 建议章节

1. **这本手册怎么用** — 故事线表 · 配图索引 · 一句话产品定位
2. **故事线 ⓪ · 开始前** — 检查清单 · 登录 · 角色默认首页
3. **故事线 ①～N** — 每条：前提 · 步骤表 · 嵌入图 · 期望结果
4. **附录** — 术语 · 常见错误 · 联系支持（可选）

## 写作原则

- 按**任务**写，不按菜单树堆砌
- 每步可观察；避免内部类名（除非交付工程师读者）
- 图与步骤 1:1；红框区域在步骤文字中点明
- 可单独发布：关键操作不依赖外链设计 doc
