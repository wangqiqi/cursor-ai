# Local rules → `.cursorGrowth/rules/local/`

**Canonical 路径**：`.cursorGrowth/rules/local/`（随 `.cursorGrowth/` gitignore）。

安装后本目录通常为**符号链接**，指向 `.cursorGrowth/rules/local/`，以便 Cursor 加载团队 `.mdc`。

请把项目特化 rules 写在 **`.cursorGrowth/rules/local/`**，不要散落到 `.cursor/rules/` 其他目录。

详见 `.cursorGrowth/rules/local/README.md`（安装后）或母版 `templates/cursorGrowth/rules/local/README.md`。
